Introduction to Cryptography

Cryptography is the practice of encoding/scrambling information so that only the intended recipient can read it. This helps protect sensitive data from unauthorized access. In Cryptography, messages are transformed from plaintext to unreadable formats using specific techniques called ciphers. Once encoded, the information can only be decoded by someone who knows how to reverse the process.

Seems simple enough, right?

There's a lot of different ways an indivudual can encode information. There's a lot of different ciphers out there, which can seem overwhelming. Have no fear, sometimes a simple Google search can help us identify what process was used.

Here are a few types of encoding:

  • Base64: A way to encode data in a format that's readable, and also reversible.
  • ROT13: A method that shifts letters 13 characters forward in the alphabet.
  • Hexadecimal: A way to present text in a numerical format, used in computing.

Let's do a Simple Walkthrough: Hexadecimal Decoding

Let's start with an easy example. Here's a message encoded in Hexadecimal:

48 65 6C 6C 6F 20 57 6F 72 6C 64 21

Each of these values represents a character in the ASCII encoding system. ASCII is a standard data-encoding format for electronic communication between computers. ASCII assigns standard numeric values to letters, numerals, punctuation marks, and other characters used in computers. Try to decode by converting each pair of hexadecimal digits into their character equivalent!

How do we go about solving this?

  1. You can Google the conversions for each piece of the encoded Hexadecimal message. For example, 48 in hex = H.
  2. Another way is to utilize online tools that decode messages. Two great resources are CyberChef and dCode. These resources allow you to input your encoded message and will generate the decoded text for you. Pretty neat, right?
