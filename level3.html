Source of the Flag

Welcome to Level 3! Sometimes, flags can simply be hidden somewhere within the website. That can mean a couple of things, so let's go over how you may be able to find a hidden flag.

  1. Use inspect element: This opens the browser's developer tools, where users can explore HTML, CSS, JavaScript, and the network requests. The HTML elements could include the flag.
  2. Browser Console: The flag could be logged to the browser console. JavaScript, which is a pogramming language use often for web development might be set to print out the flag when a certain event occurs.
  3. Check Cookies: Flags could be stored in cookies! In developer tools, by navigating to "Application", you can look for cookies set by the website. Inspect the values for hidden flags.
  4. View Source Code: By viewing the page souce, the flag could be embedded in the HTML under certain elements. You can view the page source by right-clicking anywhere on the webpage.
  5. Explore every inch: highlight the website for hidden text, look at the url, look at any media files on the page.
