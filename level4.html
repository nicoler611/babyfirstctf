Metadata Matters

To put it simply, Metadata is data about data. When you take a photo or download an image, details like the data, camera make, exposure, and more are stored within the image file itself. This is called EXIF metadata.

To extract metadata from an image, you can:

  • Use an online EXIF viewer, like Jimpl.
  • Use command-line (a text-based interface allowing users to interact with a computer) tools like exiftool or exiv2
  • Check properties of the file on your computer (Right clicking on an image, going to Properties on Windows or Get Info on macOS)
